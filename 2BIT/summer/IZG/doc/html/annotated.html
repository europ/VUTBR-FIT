Data Structures
Here are the data structures with brief descriptions:
 CBunnyVertexThis structure represents vertex that contains only position and normal
 CGPUFragmentAttributesThis struct represents fragment that is produced by rasterization. Each fragment contains fragment attributes. Each attribute is represented as array of bytes that has to be cast to requested type
 CGPUFragmentShaderInputThis struct represents input data to fragment shader
 CGPUFragmentShaderOutputThis struct represents output data of fragment shader
 CGPUPrimitiveThis struct represents primitive - output of primitive assembly unit. A primitive is composed of N vertices, interpolation type for each vertex attribute and type of each vertex attribute
 CGPUTriangleThis structure represents triangle in clip-space. This structure is used in clipping. Positions of vertices are in clip-space. Barycentric coords are in respect to original triangle (output of primitive assembly unit)
 CGPUTriangleListThis structure represents triangle list - input and output of clipping
 CGPUVertexPullerConfigurationConfiguration for vertex puller
 CGPUVertexPullerHeadThis struct represents configuration for one reading head of vertex puller
 CGPUVertexPullerOutputThis struct represents output of vertex puller - vertex
 CGPUVertexShaderInputThis struct represents input interface of vertex shader
 CGPUVertexShaderOutputThis struct represents vertex that is output of vertex shader
 CMat4This struct represents 4x4 matrix - it is represented by 4 column vectors
 CPhongVariablesThis structure contains all global variables for this method
