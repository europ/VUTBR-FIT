Detailed Description

This file contains structures and function declarations important for shader program object.

A shader program is collection of shaders (vertex shader, fragment shader). Shaders are represent by callbacks (function pointers). In common graphics APIs, shaders are compiled a programs are linked. Each program has its unique ID - ProgramID. This id can be reserved/obtained by cpu_createProgram. A program can be delete by cpu_deleteProgram. Shaders has to be attached to a program before it can be properly used on GPU. Function cpu_useProgram has to be called before any draw call can be executed.

enum InterpolationType

Type of interpolation that is done in rasterization.

Enumerator
FLAT 

flat interpolation, fragment data are generated using only first vertex of primitive

NOPERSPECTIVE 

linear interpolation without perspective correction

SMOOTH 

linear interpolation with perspective correction
